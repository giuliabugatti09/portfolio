## 🚀 Projetos em Destaque

### 🏆 [Smart Insurance: Plataforma de IA para Segurança Rodoviária](https://github.com/giuliabugatti09/AI-Analytics-Platform-for-Insurance)

Sistema end-to-end premiado para gestão inteligente de acidentes, desenvolvido em parceria com a **Sompo Seguros**.

  * **Destaque:** **3º lugar no FIAP Innovation Challenge**. Integra **Visão Computacional** (EfficientNetV2) para detecção de fotos de acidentes, **XGBoost** para predição de gravidade e um **Agente de IA (LLM)** para análise forense.
  * **Stack:** TensorFlow, XGBoost, Google Gemini API, PyDeck, Streamlit.

### [Detector de EPI com Visão Computacional](https://github.com/giuliabugatti09/PPE-Detector-CNN-)

Classificação binária de uso de capacetes em ambientes industriais para segurança do trabalho.

  * **Destaque:** Arquitetura **CNN otimizada** com foco em alta performance e baixo custo computacional.
  * **Stack:** TensorFlow, Keras, Streamlit.

### [Protegendo o Futuro do Trabalho Digital](https://github.com/giuliabugatti09/Protecting-the-Future-of-Digital-Work)

Análise do impacto da automação e IA nas ocupações globais (2024-2030).

  * **Destaque:** Uso de **K-Means Clustering** para identificar perfis de risco profissional e Random Forest para predição.
  * **Stack:** Python, Scikit-Learn, Streamlit.

### [Previsão de Churn em Cartões de Crédito](https://github.com/giuliabugatti09/bank_credit_card_churn_predicition)

Modelo preditivo para identificar clientes com propensão a cancelar serviços bancários.

  * **Destaque:** Redução estratégica de **20% nos falsos negativos**, otimizando a retenção de clientes.
  * **Stack:** XGBoost, Scikit-Learn, Plotly.

### [Sistema de Prescrição para Câncer de Mama](https://github.com/giuliabugatti09/Breast-Cancer-Prescription)

Classificação de tumores e recomendações automatizadas baseadas em dados clínicos.

  * **Destaque:** Interface interativa para suporte à decisão médica com análise de risco em tempo real.
  * **Stack:** Scikit-Learn, Pandas, Matplotlib.

### [Análise de Sentimento em Comentários](https://github.com/giuliabugatti09/sentiment-analysis-comments)

Classificação automatizada de reviews utilizando Processamento de Linguagem Natural (NLP).

  * **Destaque:** Implementação de pipelines de texto com **85% de acurácia** na classificação de sentimentos.
  * **Stack:** NLTK, Scikit-Learn, Pandas.
